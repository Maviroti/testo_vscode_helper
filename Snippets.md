# SNIPPETS
Подставляет шаблонные команды при вводе небольших тактовых последовательностей.
При наличии в шаблоне стандартных полей с параметрами можно пройтись по ним через клавишу Tab и быстро заменить дефолтные значения на необходимые
> **Например**
После подстановки `wt` появится строка `"wait "Text" timeout 60s"`.
Слово `Text` будет выделено и его можно будет заменить на необходимую строку, после этого при нажатии `Tab` выделится число `60` которое тоже можно будет быстро заменить. 


## for
`fr` > "Подставляет шаблонную конструкцию цикла for с переменной `i` для итерирования"
```
for (i IN RANGE 0 3){
    
}
```

## if
`ifc` > "Подставляет шаблонную конструкцию условия if для поиска текста на экране"
```
if (check "Text" timeout 1s){
    
}
```

`ifnc` > "Подставляет шаблонную конструкцию обратного условия if для поиска текста на экране"
```
if ( NOT (check "Text" timeout 1s)){
    
}
```

`ifs` > "Подставляет шаблонную конструкцию условия if для сравнения равенства строк"
```
if ("Text" STREQUAL "Text"){
    
}
```

`ifns` > "Подставляет шаблонную конструкцию обратного условия if для сравнения равенства строк"
```
if ( NOT ("Text" STREQUAL "Text")){
    
}
```

## Keyboard Shortcuts
`altF` > "Подставляет шаблонную конструкцию нажатия клавиш Alt + F*"
```
hold LEFTALT; press F4; release LEFTALT
```

`altT` > "Подставляет шаблонную конструкцию нажатия клавиш Alt + Tab"
```
hold LEFTALT; press TAB; release LEFTALT
```

`winR` > "Подставляет шаблонную конструкцию нажатия клавиш Win + R"
```
hold LEFTMETA; press R; release LEFTMETA
```

`winD` > "Подставляет шаблонную конструкцию нажатия клавиш Win + D"
```
hold LEFTMETA; press D; release LEFTMETA
```

## wait
`w` > "Подставляет шаблонную конструкцию ожидания wait для поиска текста на экране"
